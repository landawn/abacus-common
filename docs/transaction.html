Transaction

Transaction is supported by the Connection from Jdbc. Here is a simple sample:

final SQLTransaction tran = sqlExecutor.beginTransaction(IsolationLevel.READ_COMMITTED);
// DO NOT CLOSE the connection manually. It will be automatically closed after the transaction is committed or rolled back.
final Connection conn = tran.connection();
boolean noException = false;

try {
    // Do something with the "conn". For example:
    //
    // long id = sqlExecutor.insert(conn, INSERT_ACCOUNT, account);
    // DataSet result = sqlExecutor.query(conn, SELECT_ACCOUNT_BY_ID, id);

    noException = true;
} finally {
    if (noException) {
        tran.commit();
    } else {
        tran.rollback();
    }
}

We should avoid to open too many unnecessary transactions in heavy/big online systems.
